Etiquetas para Listas en HTML
Etiqueta Descripción
<ul> </ul> Define una lista desordenada (con viñetas).
<ol> </ol> Define una lista ordenada (numerada).
<li> </li> Define un elemento dentro de una lista (<ul> o <ol>).
<dl> </dl> Define una lista de definiciones, que muestra términos y sus descripciones.
<dt> </dt> Define un término dentro de una lista de definiciones (<dl>).
<dd> </dd> Define una descripción para un término en una lista de definiciones (<dl>).

Ejemplo visual de HTML aplicado a Listas

Conocer las etiquetas de listas en HTML es fundamental para estructurar contenido de manera clara y eficiente en una página web. Al igual que las etiquetas de estructura general (como <h1>, <p>, etc.) definen la jerarquía de un documento, las etiquetas de lista organizan información en conjuntos relacionados, ya sea de forma ordenada o desordenada.

¿Por qué son importantes las listas en HTML?

  • Mejora la legibilidad: Presentan la información de forma más clara y concisa, facilitando la comprensión por parte del usuario.
  • Facilita la navegación: Agrupan elementos relacionados, lo que ayuda al usuario a encontrar rápidamente la información que busca.
  • Aumenta la accesibilidad: Los lectores de pantalla y otros asistentes tecnológicos pueden interpretar las listas y presentar la información de manera adecuada a usuarios con discapacidad visual.
  • SEO: Las listas ayudan a los motores de búsqueda a entender mejor el contenido de una página, lo que puede mejorar el posicionamiento en los resultados de búsqueda.

Tipos de listas en HTML y sus etiquetas:

<ul> ... </ul>:
Utilizado para crear listas desordenadas, donde los elementos se presentan con viñetas (puntos).
  • Elemento 1
  • Elemento 2
  • Elemento 3
<ol> ... </ol>:
Utilizado para crear listas ordenadas, donde los elementos se presentan en un orden específico, numerado por defecto.
  1. Primer ítem
  2. Segundo ítem
  3. Tercer ítem
<li> ... </li>:
Utilizado para definir cada ítem dentro de una lista, ya sea desordenada o ordenada.
  • Ítem en una lista desordenada
  1. Ítem en una lista ordenada
<dl> ... </dl>:
Utilizado para crear una lista de definiciones, donde se agrupan términos y descripciones.
Término 1
Definición del término 1
Término 2
Definición del término 2
<dt> ... </dt>:
Utilizado para definir un término dentro de una lista de definiciones.
Término de ejemplo
Descripción o definición del término.
<dd> ... </dd>:
Utilizado para proporcionar la descripción o definición de un término en una lista de definiciones.
Término de ejemplo
