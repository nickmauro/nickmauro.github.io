Incrustar Contenido en HTML

Incrustar contenido en HTML te permite integrar elementos externos dentro de una página web. Estos elementos pueden ser videos, audios, otros documentos HTML, aplicaciones externas o incluso gráficos y animaciones interactivas.

Las etiquetas más comunes para incrustar contenido incluyen <iframe>, <embed>, <object>, así como etiquetas multimedia como <audio> y <video>.

Etiquetas Comunes

Etiquetas para Incrustar Contenido
Etiqueta Descripción Ejemplo
<iframe> Incrusta otro documento HTML o contenido web en la página. <iframe src="https://example.com"></iframe>
<embed> Incrusta contenido externo como aplicaciones o archivos multimedia. <embed src="video.mp4" width="400" height="300">
<object> Incrusta contenido como aplicaciones o documentos externos. <object data="document.pdf" width="600" height="400"></object>
<video> Incrusta un video dentro de la página. <video controls><source src="video.mp4"></video>
<audio> Incrusta un archivo de audio dentro de la página. <audio controls><source src="audio.mp3"></audio>
